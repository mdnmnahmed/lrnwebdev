UL Tag

<ul> Tag is use to use to define Unorder List in a Webpage.

UL Tag - Attributes

type: value:square,disc,circle
type attribute is use to define the type of the Unorder List.


UL Tag - Example 

        <ul type="square">
	      <li>Square
        </ul>
	    <ul type="disk">
	      <li>Disk (by Default)
	    </ul>
	    <ul type="circle">
	      <li>Circle
	    </ul>

Output

  • Square
  • Disk (by Default)
  • Circle
