Option Tag

<option> Tag is use to use to define Unorder List in a Webpage.

Option Tag - Attributes

disabled: value: -
disabled attribute is use to define that a option should be disabled when the webpage is loaded .

value: value: value of the Option element
value attribute is use to define an initial value for the option element .

selected: value: -
selected attribute is use to define that a option should automatically selected when the webpage is loaded .


Option Tag - Example 

<select>
  <option value="p1">BCA </option>
  <option value="p2">BBA </option>
  <option value="p3" selected>MCA </option>
  <option value="p4" disabled>MBA </option>
</select>
