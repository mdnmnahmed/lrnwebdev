OL Tag

<ol> Tag is use to use to define Order List in a Webpage.

OL Tag - Attributes

type: value:1,i,I,A,a
type attribute is use to define the type of the Order List.


OL Tag - Example 

        <ol type="1">
	      <li>
        </ol>
	    <ol type="disk">
	      <li>Disk (by Default)
	    </ol>
	    <ol type="circle">
	      <li>Circle
	    </ol>
