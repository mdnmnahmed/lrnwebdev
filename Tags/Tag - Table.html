Table Tag

<Table> Tag is use to use to create Table in a Webpage.

Table Tag - Attributes

border: value:pixel
border attribute is use to define a border of the table.

cellpadding: value:pixel
cellpadding attribute is use to define the space between the cell wall and the cell content.

cellspacing: value:pixel
cellspacing attribute is use to define the space between the cells.

bgcolor: value:color name,rgb(0,0,5)
bgcolor attribute is use to define the background color of the table.

width: value:pixel
width attribute is use to define the width of the table.


Table Tag contain some Tags to insert Data

<tr>: This tag is use to create Table Row.

<th>: This tag is use to insert Table Headding.
<th> tag has some attributes.They are-

Attributes Values
align : left,right,top,bottom,center
bgcolor : ColorName,Rgb(0,0,10),#000000-(Hexacode).
colspan : number
rowspan : number
width : pixel,%

<td>: This tag is use to insert Data in the Table.
<td> tag has some attributes.They are-
(see in the above <td> and <th> has the same Attributes).


Table Tag - Example 

<table border="2px" bgcolor="orange" width="500px" cellpadding="2px" cellspacing="5px">
          <tr> 
		    <th>Roll No
		    <th>S.Name
		    <th>Marks
	     </tr>
	     <tr>
		    <td>1
		    <td>Ayush
		    <td>75
	     </tr>
		 <<tr>
		    <td>2
		    <td>Noor
		    <td>85
	     </tr>
		 <tr>
		    <td>3
		    <td>Saheb

	     </tr>

		    <td colspan="2">Total

</table>
