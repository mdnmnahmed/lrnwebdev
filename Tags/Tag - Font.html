Font Tag

<font> Tag is use to use to define the font face, font size, and font color of text. in a Webpage.

Font Tag - Attributes

color: value:"Green",rgb(0,0,10)
color attribute is use to define a color of the txt.

face: value:cursive
face attribute is use to define a font_family of the txt.

size: value:20
size attribute is use to define the size of the txt.


Font Tag - Example 

<font color="green" face="cursive" size="20"> Learn HTML </font>

Output

LearnHTML
